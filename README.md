# FNF Ultimate Chrono Bot

Программное обеспечение для автоматизации игрового процесса в Friday Night Funkin' (FNF). Бот работает на основе анализа JSON-файлов чартов, что обеспечивает высокую точность нажатий вне зависимости от визуальных эффектов мода или частоты кадров.

## Основные характеристики

* **Метод работы:** Прямое чтение игровых данных (JSON), исключающее ошибки визуального распознавания.
* **Поддержка Sustain Notes:** Корректная обработка длинных нот с удержанием клавиш.
* **Локализация:** Доступны интерфейсы на русском, украинском и английском языках.
* **Адаптивность:** Стабильная работа при любой частоте кадров (60 FPS и выше) благодаря динамической системе задержки ввода.
* **Безопасность:** Автоматическая фильтрация специальных типов нот (мины, огненные стрелы), приводящих к мгновенному проигрышу.

## Установка

1. Установите интерпретатор Python версии 3.8 или выше.
2. Склонируйте репозиторий или скачайте архив с файлами.
3. Запустите файл `START_BOT.bat`. При первом запуске скрипт проверит наличие необходимых зависимостей (keyboard, colorama) и предложит их установить.

## Инструкция по эксплуатации

1. Запустите `START_BOT.bat`.
2. Укажите путь к директории `assets/data` установленной игры.
3. Выберите нужную композицию и уровень сложности из предложенного списка.
4. В процессе загрузки уровня в игре, в момент прохождения первой ноты через серую стрелку, нажмите клавишу активации (по умолчанию [1]).
5. Для принудительной остановки работы бота во время игры используйте клавишу деактивации (по умолчанию [Delete]).

## Настройка

В главном меню программы доступны следующие опции:
* **[L] Language** — смена языка интерфейса.
* **[P] Path** — изменение пути к игровым файлам.
* **[K] Keybinds** — переназначение клавиш старта и остановки бота.
* **[E] Exit** — завершение работы программы.

## Технические примечания

Синхронизация бота с игровым таймером происходит в момент нажатия клавиши старта. Для достижения максимальной точности (рейтинг Sick) рекомендуется производить нажатие точно в момент совпадения первой ноты с центром серой стрелки.
